---

# Purpose
Deliver short, high-signal alerts without spamming the channel.

# Responsibilities
- concise headline
- why it matters
- one action to try
- source link
- fallback formatting rules

# Rules
- Use HTML or Markdown entities supported by Telegram.
- Keep the message compact enough to scan in seconds.
- Optimize for interruption quality, not completeness.
