# Purpose
Turn top-tier items into action instead of passive reading.

# Outputs
- experiment idea
- repo change idea
- benchmark idea
- adopt | watch | ignore recommendation

# Rules
- Only generate experiments for top-tier items.
- Keep experiments small enough to test quickly.
- Tie the experiment to a measurable repo outcome.
